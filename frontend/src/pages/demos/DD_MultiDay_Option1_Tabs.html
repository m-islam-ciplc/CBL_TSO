Option 1: Date Tabs Approach

Concept: Each day gets its own tab. Dealer selects dates, then adds products to each day's tab.

UI Flow:

  1. Dealer clicks "Add Date" button
  2. Selects date(s) from calendar (can select multiple)
  3. Each selected date appears as a tab
  4. Click tab â†’ Add products for that specific date
  5. Submit all dates at once
