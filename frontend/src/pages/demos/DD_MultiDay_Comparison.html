Recommendation: Option 4 (Hybrid Approach)

Implementation Plan:

  1. Date Selection Component:
    • Quick buttons: "Today", "Tomorrow", "Day After"
    • DatePicker for custom dates
    • Selected dates shown as chips/tags
    • Can remove dates before submission
  2. Product Selection:
    • Same product grid as current Daily Demand
    • Click product → Popup opens
    • Popup shows quantity inputs for EACH selected date
    • Can set different quantities per date
  3. Demand Summary:
    • Grouped by date (like Option 2)
    • Shows all products for each date
    • Can edit/remove items per date
    • Shows totals per date
  4. Submission:
    • Single "Submit All" button
    • Creates multiple DD orders (one per date)
    • All orders have same order_type_id (DD)
    • Each order has its own order_id
