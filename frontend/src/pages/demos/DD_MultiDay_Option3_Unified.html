Option 3: Unified Product Selection

Concept: Select dates first, then add products. For each product, enter quantities for each selected date in a table.

UI Flow:

  1. Dealer selects multiple dates
  2. Selects products (same products for all dates)
  3. Table shows: Product | Date 1 Qty | Date 2 Qty | Date 3 Qty
  4. Enter quantities per date in table cells
  5. Submit all at once
